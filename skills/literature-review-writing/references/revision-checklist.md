# Revision Checklist

Use this checklist right before returning review prose.

## Citation Discipline

- Check whether each sentence carries one main claim.
- Split any sentence that carries multiple findings but only one citation.
- Move citations closer to the clause they support when scope is ambiguous.
- Rewrite any parenthetical citation that contains more than two references.
- Delete decorative citations that do not support a distinct claim.

## Argument Order

- Check whether the paragraph first states the relationship, type, category, or distinction.
- Check whether the paragraph then states the direction of the finding.
- Add conditions, mechanisms, exceptions, or disagreement only after the direction is clear.
- Replace vague claims such as "X influences Y" with the supported direction or pattern.

## Prose Quality

- Vary sentence length across the paragraph.
- Remove repeated sentence frames and mechanical transitions.
- Cut excess quotation marks, colons, and dashes unless they are necessary.
- Replace generic verbs with more exact verbs such as "increases", "attenuates", "predicts", "constrains", or "differentiates" when warranted by the evidence.
- Remove filler phrases that merely restate that studies exist.

## Fast Diagnostic Questions

- If I remove the citations, does each sentence still state a clear, bounded claim?
- If I inspect one parenthetical citation, is it obvious which clause it supports?
- If I inspect one paragraph, do I learn not only that a relationship exists but also what direction the literature reports?
- If I read the prose aloud, does it sound like a person explaining evidence rather than assembling templates?
