# Shared References

This folder holds repository-wide writing norms that multiple skills should reuse.

Use these files when a rule should remain consistent across literature review writing, methodology writing, data analysis writing, and theory dialogue writing.

## Files

- `citation-discipline.md`: Use when claims and citations need tighter alignment.
- `anti-ai-prose-checklist.md`: Use when prose needs a final pass to remove template-heavy or repetitive phrasing.
- `theory-dialogue-rubric.md`: Use when writing should move beyond result restatement and make a conceptual point.

## Maintenance Rule

If a guideline applies to more than one skill, put it here instead of copying slightly different versions into multiple skill folders.
