# Citation Discipline

Use this reference when a writing task needs tighter evidence-to-claim alignment.

## Core Rules

- Tie each sentence to one main claim whenever possible.
- If a sentence carries multiple claims, split it into clauses or separate sentences.
- Place citations next to the clause they support when scope might be ambiguous.
- Keep parenthetical citations lean.
- Treat more than two sources in one parenthetical citation as a signal to rewrite the sentence.

## What to Avoid

- a long sentence followed by a catch-all citation bundle
- a paragraph that ends with one citation intended to support everything above it
- decorative citations added to make a paragraph look well read
- piling up references when the prose has not distinguished the underlying claims

## Better Pattern

Move from:

"Several studies suggest that the phenomenon matters in organizations (A; B; C; D)."

To:

- state the relationship first, with a source
- state the direction or pattern next, with a source
- state the condition or disagreement after that, with a source

The goal is visible evidence discipline, not citation density.
